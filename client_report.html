General Experience and Reflections

The most challenging part of this project for me was getting past the overwhelm of designing 4 websites. Once I got organized and laid everything out in a way that was easy for me to navigate, the project became a lot easier and more manageable.

I enjoyed learning how to get past problems with the CSS not showing up on the live website. It's so satisfying to be able to problem solve and figure out what's going wrong. I also enjoyed using my previous resources about how to create an unordered list on HTML.

If I could do the project again, I would devote more time to learning how to make the website look professional and how to add more features in order to do that.

I do feel more confident in working on real-world projects after this experience because I feel like I have so many more resources and a much better understanding of how HTML and CSS work.
